# 调试指南

调试终端统一使用：

```bash
cd /home/sunrise/trash_robot_v3
set +u
source scripts/source_debug.sh
set -u
```

状态检查：

```bash
./scripts/check.sh
```

常用 topic：

```bash
ros2 topic list
ros2 topic echo /trash_target_pixel --once
ros2 topic echo /trash_target_point_camera --once
ros2 topic echo /trash_target_point_arm --once
ros2 topic echo /trash_grasp_status
ros2 topic echo /amcl_pose --once
```

常用 service：

```bash
ros2 service call /trash_grasp_once std_srvs/srv/Trigger "{}"
ros2 service call /get_pose_cmd roarm_moveit/srv/GetPoseCmd "{}"
```

停止并清理：

```bash
./scripts/stop_all.sh
```
