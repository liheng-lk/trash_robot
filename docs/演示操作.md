# 演示流程

不再保留一键大杂烩 demo 脚本。演示按模块启动，出问题时能快速定位。

## 1. WebUI

```bash
./scripts/start_web_console.sh
```

打开：

```text
http://192.168.1.121:8095
```

## 2. 导航

```bash
./scripts/start_navigation.sh
./scripts/init_pose.sh 0 0 0
```

## 3. 感知抓取

安全检查：

```bash
./scripts/start_grasp.sh dry
```

正式运行：

```bash
./scripts/start_grasp.sh live
```

## 4. 停止

```bash
./scripts/stop_all.sh
```
