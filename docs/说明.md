## 主线包

```text
trash_robot_bringup     顶层 launch
trash_robot_manager     生命周期、进程注册、系统状态、急停
trash_robot_mission     巡逻、停车、靠近、抓取、恢复巡逻
trash_robot_vision      VLM API、目标像素、深度定位、MJPEG 视频
trash_robot_grasp       手眼变换、RoArm 分类抓取
trash_robot_interfaces  自定义 action/srv/msg
trash_robot_web         Headless WebUI
base_driver             底盘串口
sllidar_ros2            雷达
roarm_*                 RoArm SDK / MoveIt 控制
```

## 运行原则

- WebUI 是控制面，硬件生命周期由 Manager 和模块脚本负责。
- 一个硬件同一时刻只允许一个 owner：相机、底盘、机械臂、导航、抓取、手眼标定不能互相抢占。
- 真实抓取只在 `start_grasp.sh live` 且机械臂服务在线时执行。
- 巡逻点必须在 AMCL 与地图对齐后记录。
- 导航调试以 RViz/RViz Web 为准，不再维护不准确的自制仿 RViz 地图作为判断依据。
- 测试按 `docs/模块启动说明书.md` 逐模块执行，不再使用一键总验收脚本。
