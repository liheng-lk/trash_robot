# 实施变更日志

## 2026-05-26 项目启动入口清理

本文件只记录当前仍有效的工程变更摘要。历史试验脚本、旧验收总入口、旧 WebUI 入口、旧视频实验入口和临时同步/Agent 辅助入口已经从 active 项目中移除，现场启动请以 `docs/模块启动说明书.md` 和 `scripts/脚本说明.md` 为准。

### 当前保留主线

- 底盘/雷达：`scripts/start_base.sh`
- 相机：`scripts/start_camera.sh`
- 机械臂运行服务：`scripts/start_arm.sh`
- 抓取：`scripts/start_grasp.sh`
- 建图：`scripts/start_mapping.sh`、`scripts/save_map.sh`
- 导航：`scripts/start_navigation.sh`、`scripts/init_pose.sh`
- 巡逻点记录：`scripts/record_patrol_waypoint.sh`
- Web 控制台：`scripts/start_web_console.sh`
- 真 RViz Web：`scripts/start_rviz_web.sh`
- 手眼标定：`scripts/start_handeye.sh`
- 手动调试：`scripts/run_base_keyboard.sh`、`scripts/start_arm_keyboard.sh`、`scripts/run_arm_keyboard.sh`
- 安全停止：`scripts/start_estop.sh`、`scripts/stop_all.sh`
- 统一入口：`scripts/trash_stack.sh`

### 当前保留抓取链路

`D435i -> VLM API -> pixel_depth_locator -> handeye_target_transformer -> roarm_sort_grasper -> RoArm services`

机械臂导航初始姿态和抓取偏移保持现场调试后的有效值，不在本次清理中回退。
