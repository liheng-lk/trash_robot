# 建图与导航

## 建图

```bash
cd /home/sunrise/trash_robot_v3
./scripts/start_mapping.sh
```

## 保存地图

地图名自己指定，不会强制覆盖旧地图：

```bash
./scripts/save_map.sh room_01
```

生成：

```text
/home/sunrise/trash_robot_v3/maps/room_01.yaml
/home/sunrise/trash_robot_v3/maps/room_01.pgm
```

## 导航

默认使用 `runtime/current_map.txt` 中记录的地图：

```bash
./scripts/start_navigation.sh
```

指定地图：

```bash
./scripts/start_navigation.sh /home/sunrise/trash_robot_v3/maps/room_01.yaml
```

启动导航时会同时启动 `trash_mission_supervisor`，WebUI 的巡逻按钮才有服务可用。

导航会复用统一相机入口，保证巡逻过程中视觉抓取链路和 WebUI 视频使用同一台 D435i。

## 初始化定位

建图起点作为原点时：

```bash
./scripts/init_pose.sh 0 0 0
```

参数含义：

```text
x y yaw_deg
```

yaw 是车头朝向角，单位是度。

## 停止

```bash
./scripts/stop_all.sh
```
