# 配置目录说明

`config/` 只放本项目运行配置，不放 ROS 包源码、不放运行日志、不放官方 SDK 配置副本。

## 分类

```text
hardware/      传感器和硬件运行配置，例如 D435i 相机
dds/           CycloneDDS 单播配置，所有脚本统一加载
navigation/    SLAM、Nav2、定位、深度转激光配置
mission/       巡逻路线和任务策略
grasp/         手眼矩阵、抓取参数、投放点、抓取偏置
perception/    VLM 感知配置与 provider registry
vision/        后续本地 BPU/hbm_runtime 后端策略，当前默认关闭
system/        RDK 资源限制、Manager 停止顺序、进程模式
web/           Web 控制台登录、安全、视频和操作配置
```

## 关键规则

- RDK 实车手眼矩阵是 `config/grasp/handeye_point.yaml`，Mac 同步时必须排除。
- `config/grasp/grasp_offset.yaml` 只做小范围现场微调，不能用来掩盖错误手眼标定。
- `config/perception/vlm_provider_registry.yaml` 只保存 provider 和模型名，不保存 API key。
- `config/vision/bpu_runtime_policy.yaml` 只定义后续本地 BPU 后端约束；当前阶段默认关闭。
- `config/dds/cyclonedds_unicast.xml` 是统一 DDS 配置；默认 `ROS_DOMAIN_ID=1`，禁止脚本回退到 FastDDS。
- 官方 SDK 包自己的配置继续留在对应 `src/roarm_*`、`src/sllidar_ros2`、`src/base_driver` 包内，不复制到这里。
